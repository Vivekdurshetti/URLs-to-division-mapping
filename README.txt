HOW TO RUN THE TOOL
====================

1. Create a folder:
   url-division-tool

2. Add these files:
   - index.html
   - style.css
   - script.js

3. Open index.html in browser.

4. Upload Excel or CSV file.

5. Paste URLs.

6. Click:
   Find Divisions
